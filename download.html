Android app

Simple requirements for mobile users

  • Android 7.0 or later
  • At least 200 MB free storage, with 500 MB recommended
  • Internet connection for sign in, sync, sharing, password reset, and AI features
  • Allow notifications only if you want reminders
Web app

Simple requirements for desktop and tablet users

  • Windows 10+, macOS 11+, recent Linux, or Android
  • Latest Chrome, Edge, Firefox, or Safari
  • 4 GB RAM minimum, 8 GB recommended
  • JavaScript, cookies, and local storage or IndexedDB enabled

Important note for users

Offline use is limited after the app has loaded once. Chrome or Edge is the best choice if you want the smoothest install and notification experience, and AI results should still be reviewed before you rely on them.
